# Sistema de Cadastro de Usuários em WPF com MVVM

Aplicação desktop desenvolvida em **C# com WPF**, utilizando a arquitetura **MVVM (Model-View-ViewModel)**, com funcionalidades de **cadastro de usuários** e **filtro de pesquisa por nome**.

Este projeto foi criado com foco em **boas práticas de desenvolvimento desktop**, organização de código e separação de responsabilidades, servindo como estudo prático de construção de interfaces gráficas em **WPF**.

---

## 🚀 Funcionalidades

- ✅ Cadastro de usuários
- ✅ Registro de **nome**
- ✅ Registro de **e-mail**
- ✅ Exibição dos usuários cadastrados
- ✅ Filtro de busca por **nome**
- ✅ Interface desktop com **WPF**
- ✅ Estrutura organizada com padrão **MVVM**

## 🧱 Arquitetura do Projeto

O sistema foi desenvolvido seguindo o padrão **MVVM**, que promove uma melhor organização do código e separação clara entre interface e lógica da aplicação.

### Estrutura conceitual:
- **Model** → representa os dados da aplicação
- **View** → interface gráfica apresentada ao usuário
- **ViewModel** → responsável pela lógica de apresentação e comunicação entre a View e o Model

### Benefícios da arquitetura MVVM:
- Melhor organização do projeto
- Código mais limpo e reutilizável
- Facilidade de manutenção
- Separação de responsabilidades
- Maior escalabilidade para futuras melhorias
